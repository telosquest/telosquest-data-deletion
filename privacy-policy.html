Dados Coletados

Data We Collect

Como Usamos os Dados

How We Use Your Data

  • Autenticação e acesso seguro à conta

Security

Todos os dados são armazenados via Firebase (Google) e protegidos com criptografia HTTPS/TLS em trânsito. Seguimos as melhores práticas de segurança recomendadas pelo Google.

All data is stored via Firebase (Google) and protected with HTTPS/TLS encryption in transit. We follow Google's recommended security best practices.
